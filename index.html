Prework Study Guide

HTML

  • The head element contains information about the webage.
  • The body element represents the visible content shown to the user.
  • Headers can be noted using h1, h2, h3, h4, h5, or h6. The text will go from largest to smallest.
  • Lists can be ordered or unordered. This means they are either bulleted, or numbered.

CSS

  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element
  • A class attribute can be used to group specific elements, and be used as a selector
  • Colours can be assigned by using semantic values, or by using a colour's hexadecimal code

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git pull origin main: pull down the latest version of the main branch
  • git checkout main: switch to the main branch
  • git clean -f: remove untracked files
